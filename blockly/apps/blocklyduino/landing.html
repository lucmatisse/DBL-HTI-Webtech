Welcome

Introduction to the tool

Welcome to the 2IOA0 Group 2 visual programming tool! This tool is tailored to the needs of children aged 10-12 who are on the autism spectrum. It aims to provide them an intuitive environment in which they can master the basics of programming by dragging, dropping, and snapping together visual blocks that represent programming concepts.

For more information on the tool, see the readme file.

To lauch the tool directly, click here!

Get started

Our tool runs completely in your web browser. You can try it straight away and use its basic features without installing anything. However, to use all features such as uploading directly to the Arduino, you will need to run a local web server.

You can download this server in the downloads section below.

Downloads

To run the program, you need to have Python 2.7 installed on your computer. If you already have this, click the left option to download only the server. If you still need to install Python, click the right option!

If you are unsure if you have the correct version of Python installed or otherwise don't know how to proceed, ask an adult for help!

Install instructions

Your download should have started! Once it is done, find the downloaded zip folder in the location you saved it to. By default, this will be your "Downloads" folder. The file you're looking for is "2IOA0 Group 2 Web Server" or "2IOA0 Group 2 Web Server + Python" if you chose to download Python too.

Once you have found the file, right-click it to open the context menu. Find and click "Extract all..." as seen in the' image on the right. A new screen will open. Here, click the "Extract" button.
You should now see a screen similar to the one shown on the left. If you downloaded Python, double-click "python-2.7.18" to launch the installer. Follow the instructions in the install wizard to install Python 2.7.

Once Python is installed, or if this was already the case, you can launch the web server. Double-clicking the file "arduino_web_server" should open it. If not, right-click it to open the context menu, choose "Open with" and select python (make sure to use version 2.7).
The file should now open a command prompt. If everything went well, the window should read: "Blocklyduino can now be accessed at http://127.0.0.1:8080/".

The setup is now complete! Leave this window open in the background and see the next section.

Launch the tool

You are now ready to launch the tool! Press this button to start learning how to program!
